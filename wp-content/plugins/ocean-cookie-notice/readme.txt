== Description ==

Add a Cookie Notice to your website and inform users you are using cookies to comply with the EU cookie law GDPR regulations.
This plugin requires the [OceanWP](https://oceanwp.org/) theme to be installed.

== Installation ==

1. Upload `ocean-cookie-notice` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins > Installed Plugins' menu in WordPress dashboard.
3. Done!

== Frequently Asked Questions ==

= I installed the plugin but it does not work =

This plugin will only function with the [OceanWP](https://oceanwp.org/) theme.
